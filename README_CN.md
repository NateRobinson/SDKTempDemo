# SDKTempDemo

这是一个帮助开发者快速上手 [ArcBlock Android SDK](https://github.com/ArcBlock/arcblock-android-sdk) 的使用的模板 Demo 项目，可以帮助开发者更快的集成 SDK 并进行相关功能开发。

## 快速开始

clone 代码到本地，导入 Android Studio 运行项目到模拟器或者真机即可。
